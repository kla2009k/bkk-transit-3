🚇 กรุงเทพฯ Transit 3D

แผนที่ 3 มิติมีชีวิตของระบบรถไฟฟ้ากรุงเทพฯ ครบ 10 สาย 193 สถานี — วางแผนเดินทาง เห็นราคาก่อนออกจากบ้าน และ "ซ้อมเดินทาง" ได้เหมือนไปมาแล้ว

🏆 Why MRTA should care

โปรเจกต์นี้ลด friction ก่อนผู้โดยสารขึ้นระบบรางจริง: ไม่รู้ขึ้นสายไหน, กลัวเปลี่ยนสาย, ไม่แน่ใจค่าโดยสาร, และกังวลเรื่องทางออก/ลิฟต์ — ทั้งหมดถูกแปลงเป็น route card, 3D preview, First Ride Simulator, MRTA Mode และหน้าที่มาข้อมูลที่ตรวจสอบได้

🎬 What judges should try

กด เพิ่มเติม → Demo Mode เพื่อเปิดโหมดกรรมการ: spotlight สาย รฟม., วางเส้นทางตัวอย่าง, เทียบสิทธิ 20 บาทกับราคาปกติ และต่อด้วยซ้อมเดินทางแบบ 3D

🚀 Next version

ต่อ official incident feed, API ค่าโดยสาร/เวลาเดินรถ, accessibility dataset ที่ครบทุกสถานี, notification เฉพาะสายที่ผู้ใช้บันทึกไว้ และ route planning ที่หลีกเลี่ยงบันไดได้

🌱 ผลกระทบที่เป็นไปได้

ระบบรางกรุงเทพฯ มีผู้โดยสารรวม >40 ล้านเที่ยว/เดือน — ถ้าเครื่องมือนี้ช่วยให้คนเปลี่ยนจากรถยนต์มาใช้รางเพิ่มเพียง 0.1% (~40,000 เที่ยว/เดือน) จะลด CO₂ ได้ราว 50-70 ตัน/เดือน (คิดจากส่วนต่าง ~125 gCO₂e ต่อผู้โดยสาร-กม. · ประมาณการ)

☀️ พลังงานสะอาดของ รฟม.

รฟม. ติดตั้ง Solar Rooftop 17 MWp ในระบบของตัวเอง ลดการใช้ไฟฟอสซิล ~21.27 ล้านหน่วย/ปี = ลด CO₂ 11,900 ตัน/ปี (เทียบเท่าปลูกต้นไม้ 680,000 ต้น) — ทุกเที่ยวที่ย้ายมานั่ง MRT คือการเดินทางด้วยพลังงานที่สะอาดขึ้นเรื่อยๆ สอดคล้องเป้าหมายประเทศ Carbon Neutrality 2050
